# Chatbot AI Sederhana (Web)

Proyek ini adalah chatbot AI berbasis web sederhana menggunakan **HTML, CSS, dan JavaScript**.
Chatbot ini dirancang fleksibel agar dapat diintegrasikan dengan berbagai penyedia AI seperti:
- ChatGPT (OpenAI)
- Gemini
- DeepSeek
- Ollama (Local)

## Cara Menjalankan
### Opsi 1: VS Code (Disarankan)
1. Buka folder project di VS Code
2. Install extension **Live Server**
3. Klik kanan `index.html`
4. Pilih **Open with Live Server**

Akses di browser:
http://127.0.0.1:5500

### Opsi 2: Langsung dari Browser
Klik dua kali `index.html`
> Catatan: untuk integrasi API AI, disarankan menggunakan Live Server.
